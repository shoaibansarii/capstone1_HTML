11) Hosting your website on Github

  1. Create new repo on Github (+)
  2. Name it, add desc, make it public
  3. Tick- Add ReadME file, and create repo
  4. 'Add file' and upload files
  5. Commit changes to add those files on Github
  6. Refresh
  7. Go to settings, Pages, Branch, change 'none' to 'main'
  8. Refresh
  9. On settings you can see a URL at 'site is live at:'
