12) HTML Tables

Syntax

 
            <table>
                //table content
            </table>

Key Elements

  • <table>: defines the table
  • <thead>: header content
  • <tbody>: body content
  • <tfoot>: footer content
  • <th>: for table heading
  • <tr>: for table rows
  • <td>: for table data/cols

Attributes

  • Rowspan <td rowspan="value">
  • Colspan <td colspan="value">
  • Caption

Example 1

        
    <table>
        <caption>Employee details</caption>
        <tr>
            <th>Name</th>
            <th>Age</th>
            <th>Designation</th>
        </tr>

        <tr>
            <td>Shoaib</td>
            <td>24</td>
            <td>Programmer</td>
        </tr>

        <tr>
            <td>Ansari</td>
            <td>25</td>
            <td>Web dev</td>
        </tr>
    </table>
