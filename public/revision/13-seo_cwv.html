13) SEO and Core Web Vitals

Core Web Vitals

  1. CLS: Cumulative Layout Shift
  2. LCP: Largest Contentful Paint
  3. FID: First Input Delay
  4. INP: Interaction to Next Paint

Use Lighthouse Report to get these values

Good SEO practices

  • Title should reflect content on page
  • Set a good title
  • Set a good URL slug
  • Set meta tags like author, title, desc
  • Compress images and other media
  • Give alt, ht, width to images
  • Remove obsolete tags
  • Add favicons
  • Add caption/titles to tables
