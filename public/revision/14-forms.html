14) HTML forms

Syntax

        
            <form>
                //form controls-->creeated by <input> tag
            </form>

NOTE: We put form controls within <div> tags as:

  • div serves as a container for HTML elements which can be easily styled using class and id attributes
  • div defines a division/section (better than the br tag)

<form 'attributes' > </form>

  1. action: sent to a server side file
  2. method: specifies the HTTP method
    • GET
    • POST

<input 'attributes' />

  1. type
  2. id
  3. name
  4. value
  5. placeholder
  6. required
  7. autofocus
  8. pattern

NOTE: Each input field must have 'name' attribute

Label Tag

<label for="id"></label>

  • Defines a label for form elements
  • Even if user clicks on label element, it toggles the form element
  • 'for' attribute should be = 'id' of input to bind together

Input types: type="_"

  • text
  • password
  • radio
  • checkbox
  • submit
  • reset
  • button
  • others: color, date, email, file, month, number, range, search, tel, time, url, week, datetime-local

Textarea element

Used when you need multiline text input like in comments, reviews.

Syntax


        <textarea name="" rows="" cols="" placeholder="optional">
            //usually left empty
        </textarea>

Select element

  • Creates a dropdown menu
  • the 'option' tag represents an item in the dropdown list

Syntax

        
            <select name="_">
                <option value="_">____</option>
                .
                .
            </select>
