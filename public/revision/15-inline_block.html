15) Block and Inline Elements

All HTML elements have specific display property

BLOCK ELEMENTS

  • starts on a new line
  • takes up entire width
  • pushes any content after them to new line

INLINE ELEMENTS

  • does not start on a new line
  • takes only required width
  • can contain other inline elements (but not block elements)
