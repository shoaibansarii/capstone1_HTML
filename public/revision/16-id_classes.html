16) ID and Classes

ID

  • unique identifier to only 1 HTML element
  • used for unique styling and JS manipulations
  • In CSS selected using #
  • Also used in linking [...url#id]--> goes directly to that id

Classes

  • for grouping elements
  • can be given to multiple elements
  • for an element, can give many classes with a space
  • used for applying same styles/behaviours to a group of elements
  • In CSS selected using .

NOTE: ID and Classes are Attributes

EXAMPLE

        
            HTML code snippet:

            <div id="firstdiv" class="red bg-yellow">First</div>
            <div id="seconddiv" class="bg-yellow">Second</div>
            <span id="firstspan" class="red">Span</span>



            CSS code snippet:

            .red{
                color: red;
            }
            
            .bg-yellow{
                background-color: yellow;
            }
            
            #firstspan{
                background-color: blue;
            }

Output:

First
Second
Span
