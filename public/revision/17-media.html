17) HTML Media

1. Video Tag

<video 'attributes'></video>

  • src: path to video file
  • controls: play/pause/volume
  • autoplay
  • loop
  • muted
  • poster
  • width, height

2. Audio Tag

<audio 'attributes'></audio>

  • src: path to audio file
  • controls: play/pause/volume
  • autoplay
  • loop
  • muted
  • preload
    • none
    • metadata (like audio duration)
    • auto

3. SVG

Three ways to embed SVG

  1. <svg></svg> tag: inline SVG, write SVG XML code in HTML doc itself
  2. using img tag and point src to an SVG file: this src can be
    • already built SVG file
    • create it by writing SVG XML code [namspace declaration required: 
      
                    <svg xmlns="http://www.w3.org/2000/svg">
                        <-- more tags here -->
                      </svg>
  3. using CSS: set SVG as background image in CSS file

4. iFrames (Inline Frames)

To embed another HTML webpage (website) on your page, alongwith external plugins and widgets

<iframe 'attributes'></iframe>

  • src: URL of page to embed (use EMBED Feature of YouTube, Google Maps)
  • height, width
  • frameborder
  • scrolling
  • name
  • allowfullscreen
