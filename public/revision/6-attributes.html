6) Attributes

  • Defines characteristics of an HTML element

  • Adds additional info to an HTML element

  • Used in the opening tag

  • Its a key-value pair of name::value

Two types of attributes:

  1. Specific to an element
  2. Global Attributes- can be used in any element
