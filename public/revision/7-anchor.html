7) Anchor Element: to create hyperlinks

Attributes

  1. 'href'
  2. 'draggable
  3. 'target' with values= _blank , _top , _self , _parent

Syntax is: <a href="target">Hi this will be blue underlined</a>

Preview: Hi this will be blue underlined

Target can be a URL or a filepath.

Anchor Element within a list element

..<li><a>hi</a></li>...

Anchor Element within a heading element

<h3><a>Go to Homepage</a></h3>

Google

Go to homepage (-blank)

An example

        
            <h3>My top 5 Favourite Websites</h3>
            <ol start="4">
                <li><a href="https://www.youtube.com/">Youtube</a></li>
                <li><a target="_blank" href="https://imranhosein.org/n/">Sheikh Imran Hosein</a></li>
                <li><a href="https://unacademy.com/">Unacademy</a></li>
                <li><a href="https://www.google.com/">Google</a></li>
                <li><a href="https://www.exposingdajjal.com/">Exposing Dajjal</a></li>
            </ol>
