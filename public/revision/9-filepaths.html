9) Filepaths

Two types of Filepaths:

  1. Absolute File path: relative to the root of the computer. Its useful in file navigation
  2. Relative File path: relative to current directory, useful in web dev
  • ../ previous directory
  • ./ current directory
  • ../../ 2 levels up

Multiple Webpages

  • Homepage/Starting file= index.html
  • other .html files in 'public' folder
  • assets folder has other files like images
